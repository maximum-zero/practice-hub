# Practice Hub

`practice-hub`는 여러 연습 프로젝트들을 하나의 형태로 관리하는 리포지토리입니다. 
Java 및 여러 기술 스택 기반의 실험/연습 프로젝트들을 통합하여 구성하며, 학습 및 구조 실험에 목적이 있습니다.

## 📢 커밋 메시지 규칙

### ✅ 기본 규칙
- 커밋 메시지는 **한글**로 작성합니다.
- 다음 형식을 따릅니다.
```text
타입(프로젝트명): 변경 내용 
```

### 커밋 메시지 타입

| 타입 | 설명 |
|----|----|
| **feat** | 새로운 기능 추가 |
| **fix** | 버그 수정 |
| **docs** | 문서 변경 (README 등) |
| **refactor** | 코드 리팩토링 |
| **chore** | 빌드 설정, 기타 작업 등 |


예시:
- `feat(java-toybox): 비밀번호 길이 검증 기능 추가`
- `fix(spring-starter): bootRun 오류 수정`
- `docs: Root README 수정`
